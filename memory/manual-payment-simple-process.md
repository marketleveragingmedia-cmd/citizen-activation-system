# Manual Payment Process - The Simple Way

## When Someone Pays Via MOSCA Marketplace, Crypto, or Wire Transfer

### Steps:

1. **Login** to Main Admin Dashboard: https://hub.citizenactivation.com/login

2. **Click** "+ Add Admin" button

3. **Fill in** their information:
   - First Name
   - Last Name
   - Email
   - Phone (optional)
   - Set temporary password

4. **Click Save**

5. **Send credentials** securely (Signal, WhatsApp, Telegram, encrypted email)

### Done ✅

They can login immediately at: https://hub.citizenactivation.com/login

---

## Payment Methods Supported:
- MOSCA Marketplace
- Cryptocurrency (BTC, ETH, USDT, MOSCA Token)
- Wire Transfer
- ACH, PayPal, Zelle, Cash App, Venmo

## Account Types:
- Team Admin: $497/year
- Organization Admin: $997 + $497/year
- White-Label: $1,997/$2,997 + $997/year

## Notes:
- All accounts created this way go directly into the same database as Stripe payments
- Accounts are immediately active
- No scripts or complex process needed
- Just use the Main Admin Dashboard interface
